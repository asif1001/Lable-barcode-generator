Barcode PDF Generator

Please upload your Excel file to the repository's uploads folder.

Once the file is uploaded, the PDF will be generated automatically by the GitHub Actions workflow.

How It Works

  1. Go to the uploads folder in your repository.
  2. Upload your Excel file there. GitHub Actions will automatically start processing it.
  3. Once processing is complete, the PDF will be available in the output folder as a downloadable PDF.

Once your file is uploaded, please wait a few minutes for the PDF to be generated.
